Week 11
ได้ลองเจนโจทย์มาฝึกเขียนพวก async await โดยมี DOM เรื่องเก่าๆเข้ามาเกี่ยวข้อง

Week12
เป็นงานจาก sprint 1 ที่ว่าเขียน table แล้ว fetch เอาข้อมูลจาก db มาใช้
ได้ฝึกใช้ DOM สร้าง element ขึ้นมาเพื่อรับค่ามาจาก db โดยที่ไม่ได้ hard code หรือ ทำเป็น static

Week13
งานจาก sprint 2 ที่มีเรื่องเวลาเข้ามาเกี่ยวข้อง ก้เลยนำเรื่องเวลามาเขียนเป้น clock.js
ที่สามารถเปลี่ยนเวลาได้ตาม UTC ถ้าหากเรา inspect sensor มันเวลาก็จะเปลี่ยนตามที่ที่เรา locate ไป

Week14
งานจาก sprint 3 ที่มีเรื่องของการ update delete ก็เลยลองสร้าง form อันนึงขึ้นมาให้สามารถ
แก้ข้อมูลหรือลบข้อมูลในตารางได้

Week15
เนื่องจาก sprint 4 จะมีเรื่องของการเช็ค period time ก็เลยเขียน อันใหม่ขึ้นมาโดยตั้งไว้ว่า
สามารถเพิ่มข้อมูลได้แค่ตอน 8 โมงเข้า ถึง 6 โมงเย็นเท่านั้น แล้วก็เขียนใหสามารถ mock time
แบบด้นๆได้หน้าเว้บเลย ถ้าเกิดอยู่ในช่วงเวลาที่เพิ่มไม่ได้ก็จะแจ้ง alert

เพิ่มเติม
Week12 - Week14 เป็นการเขียนต่อเนื่องและจะมีไฟล์ data/db.json และ myLib/fetchUtils.js
ด้วย เนื่องจาก db.json เป็นการจำลอง db ที่เหมือนว่า be ส่งมาให้แล้วในรูปของ json ผ่าน json-server
และ fetchUtils ที่ไว้ fetch และมี method ต่างๆ GET POST PUT DEL เหมือนกับว่า BE ส่ง json และ method มาจริงๆ
